# Standardized Formulas (per-100g basis)

Output convention (user standing rule): every formula on a 100 g basis, each substance with grams + %, filler q.s. to 100 g, plus a usage/dose line. Doses on a ~2.5 kg bird; avian dose-response for several actives is UNKNOWN (field-titrate).

## 1) ACUTE FOCUS BLEND (dry capsule) — per 100 g
| Substance | g/100g | % |
|---|---|---|
| L-Tyrosine | 55.00 | 55.00% |
| Sodium ascorbate (vit C, DBH cofactor) | 8.00 | 8.00% |
| Pyridoxal-5-P (B6, AADC cofactor) | 1.00 | 1.00% |
| Tribasic copper chloride TBCC (~58% Cu; DBH cofactor) | 0.104 | 0.104% |
| Rice-flour filler/flow agent | q.s. ~35.90 | ~35.90% |
| **Total** | **100.00** | **100%** |
- 1 dose = **0.5 g (1 capsule)** → delivers Tyr 275 mg, vit C 40 mg, B6 5 mg, Cu 0.3 mg.
- **Usage:** 1 capsule/bird, **45–60 min pre-fight, empty-ish crop** (tyrosine competes with dietary protein for brain uptake). Stress-gated; no effect on a calm bird. Keep DRY (Cu+vit C only react in water).
- Tyrosine 55% / 275 mg per dose is reasoned from human ~100–150 mg/kg under stress — AVIAN OPTIMUM UNKNOWN; field-titrate.

| Substance | g/100g | % |
|---|---|---|
| Dextrose monohydrate (SGLT1 fuel) | 65.57 | 65.57% |
| D-Ribose (operator field-proven energy) | 14.90 | 14.90% |
| Sodium chloride | 8.64 | 8.64% |
| Potassium chloride | 3.28 | 3.28% |
| Citric acid (acidulant) | 2.09 | 2.09% |
| L-Citrulline (free) | 3.43 | 3.43% |
| DMG | 1.19 | 1.19% |
| Sodium citrate dihydrate (trace buffer) | 0.89 | 0.89% |
| **Total** | **100.00** | **100%** |
- **Usage:** **33.55 g stick → 1 L cool water** (≈ 3.4 g / 100 mL). Make FRESH, offer through fight day, **discard same day.** Delivers per L: dextrose 22 g, ribose 5 g, Na ~53 mM, K ~15 mM, Cl ~64 mM.
- Osmolality ~291 mOsm/L (hypotonic, <300 ✓ with ribose 5 g/L). Raising ribose >~6 g/L breaches the 300 mOsm cap.
- DEB ~+3 mEq/L near-neutral. Dry stick prevents Maillard (dextrose/ribose + citrulline) until mixed; make-fresh covers the brief wet window.

## Notes

- All three are DRY (shelf ~12–18 mo, foil+desiccant); only the drink is reconstituted at use.
